# Deterministic thinned environmental data

The output object generated by running
[`thin_env_center`](https://paanwaris.github.io/bean/reference/thin_env_center.md)
on the prepared occurrence data. It contains a single calculated
centroid coordinate for each occupied environmental grid cell.

## Usage

``` r
thinned_deterministic
```

## Format

An object of class `bean_thinned_center` containing:

- thinned_points:

  A data frame of the new geometric center points

- n_original:

  The number of points before thinning

- n_thinned:

  The number of unique grid cells occupied

- parameters:

  The thinning parameters used
