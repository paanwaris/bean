# Raw Rusa unicolor occurrence data

A dataset containing the raw, unthinned occurrence records for Rusa
unicolor (Sambar deer) in Thailand. This dataset is used to demonstrate
the spatial clustering and environmental bias present in typical SDM
datasets.

## Usage

``` r
occ_data_raw
```

## Format

A data frame with occurrence coordinates:

- species:

  The species name

- x:

  Longitude coordinates in decimal degrees

- y:

  Latitude coordinates in decimal degrees

## Source

Example dataset provided with the bean package.
