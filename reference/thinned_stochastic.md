# Stochastic thinned environmental data

The output object generated by running
[`thin_env_nd`](https://paanwaris.github.io/bean/reference/thin_env_nd.md)
on the prepared occurrence data. It contains exactly one randomly
selected point per occupied environmental grid cell.

## Usage

``` r
thinned_stochastic
```

## Format

An object of class `bean_thinned_density` containing:

- thinned_data:

  A data frame of the retained occurrence points

- n_original:

  The number of points before thinning

- n_thinned:

  The number of points after thinning

- parameters:

  The thinning parameters used
